# 🛒 E-Commerce Web Application

A full-stack E-Commerce web application that allows users to register, login, view products, and add products to their shopping cart. The project is developed using HTML, CSS, JavaScript, Node.js, Express.js, and MySQL.

## 🚀 Features

- User Registration and Login
- Secure password encryption using bcrypt
- View available products
- Add products to cart
- View cart items
- Backend API integration
- MySQL database storage
- Responsive user interface

## 🛠️ Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MySQL

### Other Tools
- npm
- Git & GitHub
